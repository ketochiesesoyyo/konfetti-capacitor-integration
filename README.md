# Plus One - Wedding Matchmaking App

## Project Overview

Plus One is a wedding matchmaking app that connects singles at weddings through private, secure, and playful matchmaking. The app enables guests to swipe, match, and chat with other attendees while maintaining a fun, flirty, and respectful environment.

**URL**: https://lovable.dev/projects/0debed18-4033-4f23-be96-138199c4121e

## Key Features

- **Event Management**: Join weddings via unique invite links or create your own (paid feature)
- **Profile System**: Complete profiles with photos, prompts, interests, and bio
- **Matchmaking**: Swipe-based matching with other wedding guests
- **Liked You**: See who's interested in you and respond
- **Chat System**: Message matches and communicate with hosts
- **5 Theme Presets**: Beautiful gradients (Sunset Blush, Ocean Breeze, Golden Hour, Emerald Fizz, Midnight Rose)

## Technology Stack

- **Frontend**: React + TypeScript + Vite
- **UI Components**: shadcn-ui
- **Styling**: Tailwind CSS with custom design system
- **Routing**: React Router
- **State Management**: TanStack Query
- **Backend** (Future): Lovable Cloud (Supabase) for authentication, database, and storage

## Design System

The app uses a romantic, elegant design system with:
- **Primary Colors**: Rose gold and blush pink tones
- **Gradients**: 5 wedding-themed presets
- **Typography**: Clean, modern font hierarchy
- **Components**: Card-based layouts with smooth animations
- **Shadows**: Soft, elegant elevation system
